## Instalación y ejecución

1. Clonar el repositorio:

    git clone https://github.com/<usuario>/challenge-ticketazo.git  
    cd challenge-ticketazo

2. Instalar dependencias:

    npm install

3. Ejecutar Cypress:

- Modo interactivo: `npx cypress open`  `
